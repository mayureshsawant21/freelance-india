# Freelance India

India's dedicated freelance marketplace for Digital Marketing professionals.

## Tech Stack
- Frontend: Next.js, Tailwind CSS, ShadCN UI
- Backend: Node.js, Express, PostgreSQL
- Real‑time: Socket.io
- Payments: Razorpay
- Storage: Cloudinary

## Getting Started

### Database
1. Create a PostgreSQL database.
2. Run `database/schema.sql` then `database/seed.sql`.

### Backend
1. `cd backend && npm install`
2. Copy `.env.example` to `.env` and fill variables.
3. `npm run dev`

### Frontend
1. `cd frontend && npm install`
2. Copy `.env.local.example` to `.env.local`
3. `npm run dev`
